Micropool Setup

  1. Run the latest Bittube Daemon and allow it to fully sync.
    • Latest Bittube Daemon
  2. Obtain and enter a standard wallet address.
    • Mining directly to exchange is not supported with the micropool.
  3. Point the Daemon-Host to the IP or DNS address of the daemon.
    • Use 127.0.0.1 if the daemon is on the same machine as micropool (recommended).
    • Use the IP assigned by the router if on the same network.

Miner Setup

  1. Point the pool or connection address to the IP address of the machine running the micropool.
  2. Set the mining port to 25650.
  3. Set the username and password as preferred. These are used to identify mining rigs within the micropool.
  4. Optionally, append a number prefixed by a dot after the username (e.g. workername.10) to set the share difficulty for the rig. Difficulty defaults to 1 when left unset. As a general guideline setting the difficulty 1:1 with your miners graphrate will yield approximately 1 share every 50 seconds.
